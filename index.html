BINARY-16 FLOATING POINT CONVERTER

This simple online converter allows users to easily get the Binary-16 and Hexadecimal equivalent of any Floating-Point number representation—all special cases (NaN, Denormalize, Infinite, and Zero) included.

This is an open source project available for viewing on GitHub using the following link:

https://github.com/ghrayshell/CSARCH2-Binary16-Floating-Point-Converter
HOW TO USE

1. Choose your desired input format from the dropdown
2. Type-in your desired number (Binary or Decimal)
3. Enter your desired exponent
4. Click the “Convert Now” button
5. View the result on the output box at the bottom
6. Need the results? Save the results on your computer!
7. Reset the input values to make another conversion!
